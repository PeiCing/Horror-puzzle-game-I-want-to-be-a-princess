# 玩法

(1) 找尋五顆寶石及可逃出城堡

(2) 注意手電筒電量，並尋找電池

(3) 躲避惡管家的追逐，若被抓到則遊戲失敗

# 說明

(1) 拾獲每顆寶石皆有一段故事

(2) 惡管家離玩家越靠近，吼叫聲越明顯
